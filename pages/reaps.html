Restaurant Employees Attendance and Payroll System
This project come from a request by one of Australian Cafes manager as they need some application to make a Attendance and Payroll System. This software is made with Java and linked to Microsoft Excel and Database SQLite 3. So this software saves all of the data inside the jar file as an embedded database and can generate an excel files with multiple worksheets for each employee along with the total hour working and their wages.
